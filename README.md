# Awesome TODO CLI

A simple yet powerful command-line TODO application written in Python.

## Features

- Add, list, and remove TODO items
- Mark tasks as complete
- Persistent storage using a JSON file
- Colorful output for better readability
- Cross-platform (Windows, macOS, Linux)

## Installation

```bash
git clone https://github.com/brucestudios/awesome-todo-cli.git
cd awesome-todo-cli
pip install -r requirements.txt
```

## Usage

```bash
# Add a new TODO
python todo.py add "Buy groceries"

# List all TODOs
python todo.py list

# Mark a TODO as complete
python todo.py complete 1

# Remove a TODO
python todo.py remove 1

# Show help
python todo.py --help
```

## Requirements

- Python 3.6+
- No external dependencies (uses only standard library)
